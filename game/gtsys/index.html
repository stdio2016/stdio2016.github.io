GT-System

GT-System 是我在國中時，看到學長在做科展，科展裡面敘述的編碼系統。

GT-System 在那篇科展是用來編碼草和樹木的疊代規則用的，因為 L-System 用來描述不同長度的線段，需要重複很多次的F，因此學長發明 GT-System，以數字代替重複的F，簡化 L-System。

名稱取自草木的英文縮寫，Grass Tree System

可惜那篇科展在網路上都找不到 ;-(
