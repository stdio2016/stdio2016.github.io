stdio2016 寫的 MOD

這些是我和弟弟寫的 Minecraft MODS，我負責程式碼，弟弟負責美工 (其實很多圖案是我畫的)。 我在大學期間，為了逃離 MCreater 的魔爪，以及練習使用其他人的函數庫，我決定來用 Java 寫 Forge MOD。 這些 MOD 並沒有做很好的遊戲平衡，所以可能會太過 OP，在安裝前請小心！

以下連結會導向到 GitHub 頁面，可以到 Releases 那裡下載 MOD

Super Heated Stone (過熱石)
在這個 MOD 中，把鵝卵石熔煉成石頭之後，還可以繼續熔煉成過熱石，過熱石再熔煉成岩漿塊。 而且過熱石有特殊功能，你可以拿起過熱石隨便敲東西，然後...你試了才會知道！
支援版本：1.11.2
STrange Recipe mod (奇怪公式模組)
這個 MOD 加入了很多的公式，而且有些公式太 OP 了，會破壞遊戲平衡，所以我也不敢列出來，請自己去查原始碼，或者用 JEI 去查公式
支援版本：1.11.2 和 1.12.2
Ice Hard MOD (冰硬模組)
這個 MOD 會加入一種新的礦物，叫做「冰硬」，冰硬是一種假想的物質，會吸收所有的熱能，使得冰硬附近的水會結冰。 冰硬也可以用來製作工具，還可以用熔煉的方式來充能，使冰硬變色。
最新版本加入冰硬生態系，由於冰硬生態系充滿了冰硬，因此生態系被冷凍了，並且可以見到不同於一般生態系的植物和地表
支援版本：1.12.2
